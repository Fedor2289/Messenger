# 🔥 NexusChat — Мессенджер с ИИ-ботами

Полноценный мессенджер с поддержкой групповых чатов, каналов, отправки файлов и ИИ-ботов (Groq, OpenAI, Anthropic). Готов к деплою на Railway.

---

## 🚀 Деплой на Railway (5 минут)

### Шаг 1 — Загрузите проект на GitHub

1. Создайте аккаунт на [github.com](https://github.com) (если нет)
2. Создайте новый репозиторий (New repository)
3. Загрузите все файлы проекта в репозиторий

### Шаг 2 — Создайте проект на Railway

1. Зайдите на [railway.app](https://railway.app) и авторизуйтесь через GitHub
2. Нажмите **"New Project"** → **"Deploy from GitHub repo"**
3. Выберите ваш репозиторий

### Шаг 3 — Добавьте базу данных PostgreSQL

1. В проекте Railway нажмите **"+ New"** → **"Database"** → **"Add PostgreSQL"**
2. Railway автоматически создаст переменную `DATABASE_URL` — она подхватится сама

### Шаг 4 — Настройте переменные окружения

В Railway: **Settings** → **Variables** → добавьте:

| Переменная | Значение |
|---|---|
| `JWT_SECRET` | `любая-случайная-строка-32-символа` |
| `DATABASE_URL` | *(уже добавлена автоматически из PostgreSQL)* |

### Шаг 5 — Запуск

Railway автоматически запустит проект. Откройте URL из раздела **Deployments**.

✅ **Готово! Мессенджер работает.**

## 💬 Как пользоваться

### Создание чатов

- Нажмите **✏️** в боковой панели
- Выберите тип:
  - **Личный** — найдите пользователя по имени и начните переписку
  - **Группа** — создайте групповой чат с несколькими участниками
  - **Канал** — для публичных объявлений

### Отправка сообщений

- Напишите текст и нажмите **Enter** или кнопку **➤**
- **Shift+Enter** — перенос строки
- **📎** — прикрепить файл (картинки, документы, до 50MB)
- **Двойной клик** на сообщение — ответить на него

### Управление участниками

- Нажмите **ℹ️** в шапке чата
- Вкладка **"Участники"** — просмотр и удаление участников
- Вкладка **"Добавить"** — добавить новых участников

## 🤖 Подключение ИИ-ботов

### 1. Получите бесплатный API-ключ Groq

1. Зайдите на [console.groq.com](https://console.groq.com)
2. Зарегистрируйтесь и создайте API-ключ
3. Скопируйте ключ (начинается с `gsk_...`)

### 2. Создайте бота в NexusChat

1. В боковой панели нажмите **🤖 ИИ-Боты**
2. Нажмите **"+ Создать бота"**
3. Заполните:
   - **Имя** — отображаемое имя (например, "НексусАИ")
   - **Username** — латиницей, без пробелов (например, "nexusai")
   - **Провайдер** — Groq (рекомендуется, бесплатно)
   - **Модель** — `llama3-8b-8192` (быстрая и умная)
   - **API Ключ** — вставьте ключ от Groq
   - **Системный промпт** — описание личности бота

### 3. Добавьте бота в чат

1. Откройте нужный чат → нажмите **ℹ️**
2. Вкладка **"ИИ-боты"** → нажмите **"Добавить"**

### 4. Общайтесь с ботом

Упомяните бота в сообщении:
```
@nexusai расскажи историю про космонавта
@nexusai придумай загадку
@nexusai давай сыграем в викторину
```

Если в чате **один бот** — он отвечает на каждое сообщение.
Если **несколько ботов** — каждый отвечает только когда его упомянули.

### Поддерживаемые провайдеры

| Провайдер | Ключ получить | Бесплатно? |
|---|---|---|
| **Groq** | [console.groq.com](https://console.groq.com) | ✅ Да |

## ❓ Частые вопросы

**Бот не отвечает** — проверьте:
1. Указан ли API-ключ в настройках бота
2. Добавлен ли бот в этот чат (ℹ️ → ИИ-боты)
3. Упомянули ли вы бота через @username

**Ошибка при деплое** — проверьте:
1. Добавлен ли сервис PostgreSQL в Railway
2. Установлена ли переменная `JWT_SECRET`

**Первый пользователь не имеет прав админа** — убедитесь, что вы зарегистрировались первым. Если нет — войдите в Railway Console и выполните:
```sql
UPDATE users SET is_admin = TRUE WHERE email = 'ваш@email.com';
```
